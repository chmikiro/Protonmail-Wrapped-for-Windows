# Protonmail-Wrapped-for-Windows

Proton charges €5/month for their "desktop app" which is literally just mail.proton.me in an Electron wrapper. Same features, same everything, just €60/year for a window frame.

So I built my own. Free, open source, same functionality:

✅ Persistent login (no re-authenticating)
✅ System tray (minimize to tray)
✅ Native notifications
✅ Ad/tracker blocking built-in
✅ External links open in your browser
✅ No telemetry, no bullshit

How it works: Electron wrapper around mail.proton.me with some sensible defaults. Nothing fancy, just doesn't cost €5/month.
